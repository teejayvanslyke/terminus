YAML Extensions

YAML related extenstions.

The Kernel method yaml is a shortcut to YAML::load. 

  yaml %{
    a: 1
    b: 2
  }

produes 

  {:a=>1, :b=>2}

File.yaml? provides a way to check if a file is a YAML formatted file. 

  File.yaml?('project.yaml')  #=> true
  File.yaml?('project.xml')   #=> false
